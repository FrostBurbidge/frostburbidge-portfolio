Minor Gods

A turn based PvE game with roguelike elements in Godot engine using GDScript. The project is an in progress personal game I'm working on. I am mainly working as a programmer at present, though I also am working on the art and writing. Shown is the character ability system, which allows for easy creation of abilities due to a robust framework. The game has similar systems for characters, status effects, and levels. The process of making it is teaching me a lot about how to effectively structure large scale code to allow more effective creation of mechanics.
