Unreal Character Creator

A character creation system in Unreal Engine using Blueprints and C++. This was made for a University assignment. I was primarily doing the programming, though I also created the UI and 3D models. Shown is a segment of the blueprints for changing the parts of the character and the C++ code for saving and loading the character. This taught me a lot about working in Unreal, especially about using blueprints with C++.
