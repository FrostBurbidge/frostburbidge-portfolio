Boid Simulation

An Unreal Engine C++ simulation of bird-like objects (known as Boids). The project was made as part of a University assignment. I was primarily doing the programming, though I also created the UI and 3D models. Shown is the simulation running, with a section of the ai showing the decisionmaking process and the logic for some of the ai behaviours. I learned a lot about optimising code and ai behaviours while making this project.
